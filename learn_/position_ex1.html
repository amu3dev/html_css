Explnation

The outer div has a relative position, which means that the inner div with the profile picture will be positioned relative to the outer div. The inner div has an absolute position, which means that it will be positioned relative to the outer div. The top and right properties are used to position the profile picture in the top-right corner of the card.

The outer div (the card) is positioned relative, so the inner div with the profile picture (positioned absolute) will be 10px from the top-right corner of this card, not the whole page.
